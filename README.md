# Mechaboy (E-commerce MERN app)

Mechaboy is an e-commerce MERN application, built with ShadCN UI and Redux Toolkit.

## Overview

Mechaboy is an online store, selling mechanical keyboards and keyboard parts. 

<p align="center"><img src="https://github.com/thalangi33/mechaboy-ecommerce/assets/99813905/019de651-0151-4853-a641-f83a43df9d41" width="70%"></p>

The frontend is built with:
- React Typescript
- Tailwind CSS
- ShadCN UI components
- Redux Toolkit (State management)
  
The backend uses:
- Mongoose (Database)
- Express.js (API calls)
- Stripe to handle payments.
